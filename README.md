# attendance-manager

An employee attendance management program for a company with approximately 40 regular employees and about 10 outsourced employees for monthly events. The program needs to track attendance, manage schedules, and possibly handle different categories of employees (regular and outsourced).

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/attendance-manager.git
cd attendance-manager
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
